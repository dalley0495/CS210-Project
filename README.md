CS 210 Portfolio Submission ? Corner Grocer Project
Project Summary

The Corner Grocer project is a program that reads grocery purchase data from a file and tracks how often each item is purchased. The program allows the user to search for specific items, display all item frequencies, and view a histogram of purchase data. This solves the problem of organizing and analyzing store purchase information in a clear and efficient way.

What I Did Particularly Well

I did well at organizing my code and making the program function correctly. I was able to successfully read data from a file, use a map to store item frequencies, and display the results in multiple formats. I also created a working menu system that allows users to interact with the program easily.

Where I Could Enhance My Code

I could improve my code by adding better error handling, such as checking if the input file opens correctly and validating user input. I could also improve efficiency by reducing repeated code and making the program more modular. These changes would make the program more reliable, easier to maintain, and more secure.

Challenges I Faced and How I Overcame Them

The most challenging part of this project was learning how to use maps in C++ and understanding how to properly read and process data from a file. I overcame this by practicing, reviewing examples, and using resources like ZyBooks and Google. These tools helped me better understand how the code works and how to fix errors.

Transferable Skills

This project helped me develop skills in file input/output, using data structures like maps, debugging, and problem-solving. These skills are important for future programming projects and coursework because they are commonly used in real-world software development.

Maintainability, Readability, and Adaptability

I made the program maintainable and readable by using clear variable names, adding comments, and organizing the code into logical sections. This makes it easier to understand and update the program in the future. The structure of the code also allows for changes or additional features to be added without major issues.
